## 문제 정의

### 한국 SME의 해외 진출 3대 장벽

#### 1. 정보 접근성 부재 (Information Gap)

```
문제: EU/US 입찰 정보가 45+ 플랫폼에 분산
  - TED (EU): 영어/불어/독어
  - SAM.gov (US): 영어 전용
  - UN Procurement: 복잡한 포털

결과: 85%의 EU/US 입찰 정보를 한국 SME가 모름
비용: 연간 $50M+ 기회 손실 (중견기업 기준)
```

#### 2. 언어 장벽 (Language Barrier)

```
문제: 영어 조달 문서 번역에 평균 6-8시간 소요
  - 구글 번역: 전문 용어 오역
  - 전문 번역: 건당 $500-1,000

결과: 번역 비용 부담으로 입찰 포기
비용: 연간 $20M+ (번역 외주 비용)
```

#### 3. 제안서 작성 부담 (Proposal Cost)

```
문제: 영문 제안서 작성에 $2,000-5,000 컨설팅 비용
  - 전문 작성: 2-3주 소요
  - 낙찰 불확실: ROI 낮음

결과: 월 최대 2-3건만 도전 가능
비용: 놓친 고가 계약 연간 $100M+
```

### 정량적 영향

| 항목                       | 현재 상황       | 연간 비용           |
| -------------------------- | --------------- | ------------------- |
| EU/US 입찰 미발견          | 85% 놓침        | $50M (기회비용)     |
| 영어 번역 외주             | 건당 6-8시간    | $20M (번역 비용)    |
| 제안서 작성 컨설팅         | 건당 $2K-5K     | $10M (외주 비용)    |
| **총 비용 (중견기업 기준)** | -               | **$80M/년**         |

## 솔루션

### BIDFLOW의 핵심 가치

#### 1. 글로벌 입찰 자동 발굴 (Automated Discovery)

**연동 데이터 소스 (45+)**:

| 지역       | 플랫폼                       | 연간 규모        |
| ---------- | ---------------------------- | ---------------- |
| **EU**     | TED (Tenders Electronic Daily) | $2.8T            |
| **US**     | SAM.gov, FedBizOpps          | $500B            |
| **아시아** | Singapore GeBIZ, HK eTender  | $100B            |
| **국제**   | UN Procurement, World Bank   | $50B             |
| **한국**   | 나라장터, G2B                | $150B            |

#### 2. 175점 AI 매칭 엔진

1. **키워드 매칭** (100점): 강한 키워드 + 약한 키워드 - 제외 키워드
2. **규격 매칭** (25점): 제품 스펙 자동 추출 및 비교
3. **기관 매칭** (50점): 45개 주요 발주기관 정규화 및 선호도

#### 3. AI 스마트 함수 (5종)

| 함수               | 기능                    | 가치               |
| ------------------ | ----------------------- | ------------------ |
| **=AI_SUMMARY()**  | 공고 2-3문장 요약       | 읽기 시간 90% 단축 |
| **=AI_SCORE()**    | 낙찰 가능성 0-100%      | 우선순위 명확화    |
| **=AI_MATCH()**    | 최적 제품 추천          | 오판 방지          |
| **=AI_KEYWORDS()** | 핵심 키워드 3개         | 빠른 스캔          |
| **=AI_DEADLINE()** | 마감일 분석 + 일정 제안 | 미스 방지          |
